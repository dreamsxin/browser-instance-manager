## 使用方法

### 安装依赖
```bash
npm install axios @faker-js/faker cli-table3
```

### 基本使用
```bash
node throughput-test.js -c 10 -d 20000

# 请求数模式 (执行100个请求)
node throughput-test.js -c 10 -n 100

# 持续时间模式 (运行60秒，推荐用于吞吐量测试)
node throughput-test.js -c 20 -d 60

# 高并发测试
node throughput-test.js -c 50 -d 120

# 自定义服务URL
node throughput-test.js --url http://192.168.1.100:3000 -c 10 -d 30
```
